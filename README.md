# Dofus_optimisateur_stuff
Un projet d'optimisation de stuff dofus

Note : tout le projet est en français, et il est utile au jeu Dofus qui est joué principalement par des français. 
Donc j'écris en français. 

Ce projet a pour but de permettre de trouver un stuff Dofus pour un personnage. 
Le programme fonctionne (sur Windows). 

Une version utlisable du programme se trouve dans le fichier "opti_stuff_dofus_2_1.zip" : 
Pour l'utiliser : 
1. Télécharger ce fichier
2. Le dézipper
3. Lire "Informations.txt" et lancer "opti_stuff_dofus.exe"

Le code se trouve dans les différents autres fichiers et dossiers.
